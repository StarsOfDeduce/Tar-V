# Tar V - 极简而高级的 Windows 快捷启动面板

Tar V 是一款专为 Windows 系统打造的**高档毛玻璃拟物风快捷启动面板（Launcher）**。它致力于将杂乱无章的桌面快捷方式收纳进一个轻量、优雅、高动态响应的面板中。

通过设置全局快捷键（默认 `Alt+Q`），您可以一键呼出/隐藏面板，并支持通过鼠标悬停、滚轮滚动或数字键快速在不同分组标签页之间自由切换。

---

## ✨ 核心特性

- 🪟 **高级毛玻璃拟物美学**：精心调配的深色/浅色/跟随系统主题，支持背景透明度与面板圆角大小微调，搭配极富动感和细腻阴影的微交互动效。
- 📂 **原生系统图标提取**：完全保留 Windows 本地文件、文件夹、快捷方式本身的图标，不做多余滤镜与二次修改，保留最纯粹、最熟悉的分辨体验。
- 🖱️ **拖拽一键添加**：支持直接将文件、文件夹、网络 URL 或桌面快捷方式拖入面板网格中一键添加卡片，支持自动解析 Windows `.lnk` 目标路径与运行参数。
- 📦 **快捷批量管理**：支持进入“批量管理模式”，进行多选、全选，批量删除、批量修改运行参数/管理员属性，或一键跨分组批量移动图标。
- 🛠️ **高级维护服务**：内置一键修复并重建全局图标缓存、重复快捷项清理、空标签删除，以及一键扫描并清理失效的本地死链。
- ⚙️ **独立设置中心**：与主面板完全独立的设置窗口，支持滑块微调字体大小（标签字号、卡片字号、设置字号独立缩放），设置页面调整大小时具有永久自动平滑居中功能。
- 💬 **定制弹窗与选单**：全局所有下拉框、文本输入框、确认对话框已实现深度美化（消除 Windows 原生直角和蓝底背景），错误路径时自动换行，长文本带内部滚动，保证视觉完美统一。

## 🚀 开发者指南

如果您希望从源码运行或二次开发 Tar V，请参照以下步骤：

### 1. 克隆/下载本仓库到本地

```bash
git clone https://github.com/您的用户名/tar-v.git
cd tar-v
```

### 2. 安装项目依赖

确保您的系统中已安装了 [Node.js](https://nodejs.org/) (建议使用 LTS 版本)。然后在项目根目录下运行：

```bash
npm install
```

### 3. 开发模式下运行

执行以下命令直接拉起 Electron 开发窗口进行实时调试：

```bash
npm start
```

### 4. 编译打包生成 Windows 便携版或安装程序

项目默认配置为输出带有安装向导的 **NSIS 安装包**（可自动创建桌面和开始菜单快捷方式并支持系统卸载）。执行以下命令开始打包：

```bash
npm run dist
```

打包完成后，输出文件将保存在根目录下的 `dist/` 目录中：
- 安装包：`dist/Tar V Setup 1.0.0.exe`
- 免安装解压版主程序：`dist/win-unpacked/Tar V.exe`

*(若需要单文件便携绿色版，可在 `package.json` 的 `build.win.target` 中将 `"nsis"` 修改为 `"portable"` 重新打包)*

## 🎨 界面视觉定制

您可以通过 **设置中心 -> 外观** 来实时调整：
- **界面风格**：深色模式 / 浅色模式 / 跟随系统
- **网格列数**：支持 6 列、7 列、8 列、9 列、10 列、12 列布局切换
- **背景基色与高亮主题色**：配备调色盘，可无缝更改品牌橙红主题色或基础不透明度
- **字体大小微调**：拥有三个独立滑块，可随时更改主页分组标签、快捷图标名称、设置页内容字体大小，以适应不同的屏幕 DPI 分辨率。
